## 📥 Installation & Setup  

### 📂 Clone the Repository  
```sh
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

### 🚀 Running the Project  
1️⃣ Install dependencies:  
   ```sh
   npm install
   ```  
2️⃣ Start the development server:  
   ```sh
   npm start
   ```  
3️⃣ Open `http://localhost:3000` in your browser 🌐
